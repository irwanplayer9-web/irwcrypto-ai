🚀 CARA CEPAT BUILD APK:

  METODE 1: EAS BUILD (TERMUDAH)
  ─────────────────────────────────────
  1. npm install -g eas-cli
  2. eas login
  3. eas build:configure
  4. npm run build
  5. Download APK dari expo.dev
  
  METODE 2: TESTING DEVELOPMENT
  ─────────────────────────────────────
  1. npm start
  2. Scan QR code dengan Expo Go app
  
  METODE 3: LOCAL BUILD
  ─────────────────────────────────────
  1. npm run prebuild
  2. cd android
  3. gradlew.bat assembleRelease
  4. APK di: android/app/build/outputs/apk/release/
