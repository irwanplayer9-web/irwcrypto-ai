🔧 STEP 3: CONFIGURE PROJECT

Setelah login berhasil, run:
  eas build:configure

Jawab pertanyaan:
  - Platform: Android (pilih Android)
  - Proceed: Yes

File eas.json sudah ada, jadi ini akan update project ID.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🚀 STEP 4: BUILD APK!

Run:
  npm run build

ATAU:
  eas build --platform android --profile production

Proses:
  ✓ Upload project files
  ✓ Install dependencies
  ✓ Compile Android app
  ✓ Generate APK

Waktu: ~10-15 menit

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📥 STEP 5: DOWNLOAD APK

Setelah build selesai:

1. Terminal akan show link ke expo.dev

2. Buka link di browser
   ATAU
   Buka: https://expo.dev/accounts/[username]/builds

3. Klik build terbaru

4. Klik tombol "Download"

5. Save APK file (sekitar 50-60 MB)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📱 STEP 6: INSTALL DI HP ANDROID

1. Copy APK file ke HP Android
   (via USB, email, cloud, WhatsApp, dll)

2. Di HP, buka File Manager

3. Tap file APK

4. Jika muncul "Install from unknown sources":
   - Tap "Settings"
   - Enable "Allow from this source"
   - Tap Back
   - Tap "Install"

5. Tunggu install selesai

6. Tap "Open"

7. ENJOY! 🎉
   - Lihat animated splash screen
   - Test mining features

⚠️ TROUBLESHOOTING:

Issue: "eas command not found"
  → Close & reopen terminal
  → Or run: npm install -g eas-cli

Issue: Login failed
  → Check username & password
  → Check internet connection
  → Try: eas whoami

Issue: Build failed
  → Check build logs di expo.dev
  → Run: eas build --platform android --profile production

Issue: Can't install APK
  → Enable "Install from unknown sources"
  → Settings → Security → Unknown Sources
