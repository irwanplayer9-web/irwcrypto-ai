# Assets Folder

Please add the following image files to this folder:

1. **icon.png** - App icon (1024x1024 px recommended)
2. **splash.png** - Splash screen image (200px width recommended)
3. **favicon.png** - Web favicon (48x48 px)

You can generate these images or use placeholder images for testing.

For testing, you can download free icons from:
- https://www.flaticon.com/
- https://icons8.com/
- Or use online icon generators
