## 🚀 Build APK Process

### After replacing icons:

```bash
# Method 1: EAS Build (RECOMMENDED)
npm install -g eas-cli
eas login
eas build:configure
npm run build

# Wait 10-15 minutes
# Download APK from expo.dev

# Method 2: Test First
npm start
# Scan QR, test splash animation
# Then build with Method 1

# Method 3: Local Build
npm run prebuild
cd android
gradlew.bat assembleRelease
# APK at: android/app/build/outputs/apk/release/
```
